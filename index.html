Projects

Becoming a Certified NHD Editor

In Spring 2019 I landed an internship at the National Park Service under the GIS coordinator Dave Best. Our task was to clean up a stream layer that was derived from LiDAR and a flow-accumulation model. When the remote sensing vendor delivered the product it was clear that it was not going to be immediately useful; we needed to sort through all of the excess lines. When that menial task was completed Dave decided it would be great to input the data into the National Hydrography Dataset. A couple months and a few trainings later I'm officially allowed to input data into the NHD!

Using Python to Process LS 8 images

Using the image manipulation module Rasterio I wrote a script that took a decompressed LS 8 level 1 data packet, radiometrically corrected it and brought it to top-of-atmosphere reflectance. Me and my partner then did some raster math using the different bands to create Vegetation indices automatically. We used the QT GUI module to create an easy to use interface. Learning how to combine various Python packages and manipulate images as arrays was incredibly rewarding and pretty much solidified python as my favorite data processing language.
Looking back on this code months later was an interesting experience; now that I've gotten better at using Python I realised I should have done so many things differently... Click here to view the code on my Github.

Building Out a Geodatabase

Just recently I was hired as a Research Assistant at HSU's Cultural Resource Facility to help the GIS people there organize decades of survey data. Their final goal is to mimic the Northwest Information Center's schema so that data can be more easily interchanged between various organizations that the CRF works with. As of now we are doing the grunt work of collecting all the legacy data and making a shell database in Microsoft Access.


California Groundwater Ambient
Monitoring and Assessment Program


I had never made a map gif before so I figured I'd start with something fun... The way that groundwater is tested and estimated has always interested me. So I visited the CA Open Data Portal and downloaded the most recent GAMA dataset that the California Water Resources Board had. I extracted from the dataset the samples of Uranium, Arsenic, and Nitrate that exceeded the EPA's recommended safe levels and presented them as symbols on this map gif; the size of the circle on the map is representative of the sample size.

The samples were not taken at the same wells every year which is unfortunate but this simple gif can help visualize ground-water quality trends around the state. Click here to learn more about the GAMA program.
